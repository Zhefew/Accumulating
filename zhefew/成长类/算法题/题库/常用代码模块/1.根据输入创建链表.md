标准答案
```
	ListNode* createList(const vector<int>& nums) {
	    if (nums.empty()) return nullptr;
	    ListNode* head = new ListNode(nums[0]);
	    ListNode* cur = head;
	    for (int i = 1; i < nums.size(); ++i) {
	        cur->next = new ListNode(nums[i]);
	        cur = cur->next;
	    }
	    return head;
	}
```
```
ListNode* createList(const vector<int>& nums)中的const的作用在于
const使得传入的num数组无法修改
参数类型	                    是否可读	是否可修改	是否拷贝数据
vector<int>（传值）      	   ✅	  ✅	          是
vector<int>&（普通引用）	       ✅	  ✅	          否
const vector<int>&（常量引用）  ✅	  ❌	          否
```
